# SchoolNet — Marketing Website

This folder contains the static marketing website for **SchoolNet**, the school management and learning platform.

**Built by [Orion13 Technologies](https://orion13.us)** · [orion13.us](https://orion13.us)

## Contents

- **index.html** — Landing page (hero, features, who it’s for, benefits, CTA, footer)
- **project.html** — Project structure and technical documentation
- **styles1.css** — Layout and styling (responsive)
- **script.js** — Mobile menu toggle

## Run locally

Open `index.html` in a browser, or serve the folder with any static server, for example:

```bash
# From project root
npx serve website
# or
python -m http.server 8080 --directory website
```

Then visit the URL shown (e.g. http://localhost:3000 or http://localhost:8080).

## Deploy

Upload the contents of `website/` to any static host (e.g. Netlify, Vercel, GitHub Pages, or your own server). No build step required.
